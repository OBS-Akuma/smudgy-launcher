smudgy client
Launcher
Smudgy Client is a custom Electron-based Kirka.io client focused on advanced UI customization, player cosmetics, quality-of-life features, and social enhancements. It includes animated gradients, custom badges, lobby news, Discord integration, spectate tools, friend pinning, profile customization, CSS theming, in-game HUD improvements, and script/plugin support for a fully personalized experience.

Frequently Asked Questions
Is Smudgy client safe?

Yes, Smudgy client is safe because it's open source, has been helped on by many users, and is used by many users.

Does this client use my token?

Yes, we use your token for API calling only for the official Kirka API.

Can I have a custom badge?

No, they are given out at random or from giveaways. If you ask, you will be blacklisted.

Will I get banned for using Smudgy client?

Not that I'm aware of. If Xip flags something, I will take action.
